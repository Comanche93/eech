Here is a list of the sounds that are used to build the sound of player-controlled helicopters:


aprotore.pcm	Apache rotor sounds, normal flight, exterior
aprotores.pcm	Apache rotor sounds, excessive maneuvering ("rotor slapping"), exterior
aprotoret.pcm	Apache turbine sound, exterior
aprotori.pcm	Apache rotor sounds, normal flight, interior
aprotoris.pcm	Apache rotor sounds, excessive maneuvering ("rotor slapping"), interior
aprotorit.pcm	Apache turbine sound, interior

harotore.pcm	Havoc rotor sounds, normal flight, exterior
harotores.pcm	Havoc rotor sounds, excessive maneuvering ("rotor slapping"), exterior
harotoret.pcm	Havoc turbine sound, exterior
harotori.pcm	Havoc rotor sounds, normal flight, interior
harotoris.pcm	Havoc rotor sounds, excessive maneuvering ("rotor slapping"), interior
harotorit.pcm	Havoc turbine sound, interior

The sounds for the Blackhawk, Comanche, Hind and Hokum use a different naming scheme that does not need much explanation, 
since the longer filenames basically explain themselves.

All these files should go into the cohokum\audio\ directory, right next to the game's original sound.dat and sound.hdr

NOTE: You can also put modified versions of the game's built-in sound effects into that same directory.
The sound files should be 22050Hz, 8bit, Mono.
EECH will detect this and use your modified sound files instead of the original ones, so you do not have to edit the contents of sound.dat anymore.
To find out the filenames, use a tool like GSS to extract the filenames, or take a look at this list:
