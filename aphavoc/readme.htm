INSTALLATION

Simply copy the file cohokum.exe into your /razorwroks/cohokum directory. You may want to rename your old cohokum.exe to save it.

What 's NEW in EECH ?

eech.ini parameter
Description
Downwash effect
dwash=1

rotor downwash (dust) effect. It makes you see clouds of dust when the chopper gets close to the ground! (default =1, on)
Mouselook and TrackIR support

msl=0
msls=15

You can now control you view using the mouse or by using TrackIR. The TrackIR support uses enhanced mode which means that you will not have to bother about centering you view all the time. A check will be made for a TrackIR upon startup, and if it is found it will be used to control the view. Otherwise the mouse is used. (default = 0, off)

Pressing the left mouse button will recenter the view to forward.

You can set the speed of the mouselookwich should be a value greater than 0 (default = 15).

Field of View

minfov=20
maxfov=80

The user can specify different field-of-view values, which can be activated by the '7','8' and '9' keys. 7 switches min FOV, 8 switches to (fixed) standard FOV, 8 switched to wide FOV. Please note that higher FOVs currently mean lower LODs.

FLIR panning and zoom (Comanche only)

eopann=-1
eopanv=6
eopanh=8

eozoomn=-1
eozoomax=7

You can now use joystick axis to pan the FLIR display. You specify the joystick number and which axis to use for this feature.

The old FLIR code only allowed three levels of zoom. This code has been updated to support any zoom levels. When using the keyboard, you still use the same keys to zoom, but you now have 5 levels instead. If you have an extra joystick axis available you can map one of them to the zoom and using for zooming.

2  Y-axis
3  Z-axis
4  rx-axis
5  ry-axis
6  rz-axis
7  slider 1

Mappable joystick axes

cyclicn=-1
cyclich=1
cyclicv=2

collectiven=-1
collectiveax=3

ruddern=-1
rudderax=6

You can now choose exactly which joystick and which axis on that joystick that should be used for cyclic, collective and throttle.

Ground stabilised FLIR (Comanche only)

grstab=1

The Comanche now supports ground stabilisation for the FLIR. You toggle this feature in flight using Control-S.
Infantry

radarinf=1

Infantry does not show up on ground radar anymore
Infantry can never be seen on the ground radar now. Default = 1 (on). To restore set to 0.

Fly any aircraft

faa:1

Fly any aircraft: all helicopters available now for the user. Fly your own insertions. The cockpit defaults to the Apache when choosing anything else than CHAH (default = on)
Wideview cockpit

commanche pilot=x,y,z
commanche co-pilot=x,y,z
hokum pilot=x,y,z
hokum co-pilot=x,y,z
apache pilot=x,y,z
havoc pilot=x,y,z

Retreating the pilots head causes the player to have a wider field of view without distortion. No more F1 / F2 pressing! The values can be edited by pressing Ctrl-\ in the game and using the numpad keys to move the viewpoint: 4 and 6 distance to dashboard; 2 and 8 vertical; 1 and 3 horizontal. Alt-\ quits editing and saves the values in the eech.ini file. (Note: the file widemod.cfg is no longer needed).

Note: for some keyboards the keys are not Ctrl-\ and Alt-\ but something else. You just have to try for yourself on the right hand side of the keyboard. For more info: http://home.planet.nl/~jette073
WUT
wut=[filename]

WUT = Weapons and Units Tweaker by DeStRo (Daniel Blazevic) http://www.aplus.ba/dblazev/wut/, a program to tweak loadout and behaviour of all 'entities' in the game. Most multiplayer games are done with wut 1.5.1D a series of reality tweaks that make life nice and difficult in EECH. With the WUT program you can save all your tweaks in a text file. The name of this file can be added to the eech.ini (e.g. 151_D.txt). More info here: http://home.planet.nl/~jette073
eech.ini
automatically generated at startup

All game parameters (command line) in one file with a short description. You can edit the parameter values if you leave a blank space between a value and the comment '#'.
Fly any aircraft

faa=1

Fly any aircraft: all helicopters available now for the user. Fly your own insertions. The loadout is correct for the helicopter type but unchangeable. The cockpit defaults to the Apache when choosing anything else than C-H-A-H, default = 1 (on)
Show framerate
dfr=0

The framerate is displayed in the top left corner and updates about once a second. The average of 30 readings is shown for stability. Values: 0 = off; 1 = on; 2 write the framerate to a file framerate,txt in the cohokum directory
Multiplayer master server

At http://hox.dhs.org/eech there is a masterserver for EECH, listing all running games along with total players and IP addresses. EECH versions newer than 1.6.0W1 report to the masterserver if you play a game with Networking set to TCP/IP in the options menu. To join a game, just copy&paste an active IP from the serverlist into GSS (Ground Support System) or EECH itself.

High resolution MFD
highresmfd=0

The resolution of the Comanche and Hokum MFD's will be doubled. Remember that there is a small framerate hit when activating this feature.

TSD render mod
tsdrender=1
tsdpalette=2
tsdenemy=0

This mod adds a relief map background to the TSD of the Comanche, Hokum and Apache (the Havoc is not included yet). The map is shown as elevation colours in several colour palettes: green/yellow/brownish colours and water is blue. The maximum elevation of the map is used to scale the colours. Shaded relief is also added as an option.

- The Apache now has the same TSD as the Comanche.
- Optional: the friendlies and enemies are shown in red and blue colours (including detection circles of SAMs). If you don't want this option set the eech.ini variable "tsdenemy" to 0.
- The configuration is automatically saved in the eech.ini file when you leave the game.
Pressing Ctrl-D (as in Declutter) cycles through the options: old TSD screen, the terrain map with/without contours, shaded relief with/without contours. Alt-D cycles through a 3 colour palettes.

 

SOURCE CODE and VERSION NUMBERS

If you want to give the source code a try you can find all the necessary details at the EECH DevZone: http://eech.bollocks.net.nz. You can aslo subscribe (free) to a developers mailing list.

  • The versions are named as follows: for instance 1.6.0W8 means
  • 1 = major version (will probably never change)
  • 6 = data version, was officially 4, then the weapons and units tweaks created a version 5 and the source code version has been called 6
  • 0 = minor version number, currently at 0, will be chnaged to 1 when the first stable version is released
  • W = build type: 'C' is commercial; 'X' means experimental release, 'W' compiled with Watcom Open Source, 'V' means compiled with Visual C
  • 8 = developers number

Fixes/features in 1.6.0W7:

  • primary and secondary masterserver ini settings & command line options work. New default values added.
  • Fly any aircraft fixes.

  • random task assignments that mess up the gameplay fixed
  • small eech.ini fixes that caused crashes (please remake eech.ini)
  • keyboard behaviour back to normal
  • TrackIR not included

  • "eech.ini" use included
  • downwash effect included
  • TrackIR not included
  • multiplayer buggy
